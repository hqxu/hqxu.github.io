MarkdownPad is a full-featured Markdown editor for Windows.

Give them a try:

  • Bold (Ctrl+B) and Italic (Ctrl+I)
  • Quotes (Ctrl+Q)
  • Code blocks (Ctrl+K)
  • Headings 1, 2, 3 (Ctrl+1, Ctrl+2, Ctrl+3)
  • Lists (Ctrl+U and Ctrl+Shift+O)
